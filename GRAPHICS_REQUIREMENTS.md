## üì± Screenshot Capture Instructions

### Using Android Device:
1. Open the app
2. Navigate to desired screen
3. Press **Power + Volume Down** simultaneously
4. Screenshot saved to Gallery
5. Transfer to computer for editing

### Using Android Emulator:
1. Open Android Studio Emulator
2. Run the app
3. Click camera icon in emulator toolbar
4. Or use **Ctrl + S** (Windows) / **Cmd + S** (Mac)
5. Screenshot saved automatically

### Editing Screenshots:
- **Tools:** Canva, Figma, Photoshop, GIMP
- **Add:** Text overlays, highlights, arrows
- **Resize:** To required dimensions
- **Optimize:** Compress if file size is large

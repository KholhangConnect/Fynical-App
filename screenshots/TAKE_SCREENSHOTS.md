### Step 1: Run App in Emulator

Follow this order to capture all required screenshots:

1. **Home Screen**
   - App opens to home screen
   - Shows all calculator categories

   - Go back to home

   - Show totals
   - **Screenshot:** Press `Ctrl + S`

Screenshots are saved in:
```
C:\Users\YOUR_USERNAME\Pictures\Screenshots\
```
Or check Android Studio's screenshot location.

## üì± Method 2: Using Physical Android Device

### Step 1: Install App on Device
1. Build and install APK on your Android device
2. Or use USB debugging to install

### Step 2: Take Screenshots
1. Open the app
2. Navigate to each screen
3. Press **Power + Volume Down** simultaneously
4. Hold for 1-2 seconds
5. Screenshot saved to Gallery

### Step 3: Transfer to Computer
1. Connect device via USB
2. Copy screenshots from `DCIM/Screenshots/` folder
3. Or use Google Photos/Drive to sync

6. Export as PNG

3. Add text, arrows, highlights

## üÜò Troubleshooting

### Screenshots Too Small
- Use higher resolution emulator/device
- Minimum: 720 x 1280 pixels
- Recommended: 1080 x 1920 pixels

### Screenshots Blurry
- Ensure app is fully loaded before screenshot
- Use high DPI emulator/device
- Don't zoom in/out before screenshot

### Can't Find Screenshots
- Check Android Studio's screenshot folder
- Check device's Gallery/Screenshots folder
- Use file manager to search for PNG files

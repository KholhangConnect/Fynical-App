Privacy Policy for Fynical

Last Updated: January 2026

Introduction

Fynical ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our mobile application Fynical (the "App").

Information We Collect

No Data Collection

Fynical does NOT collect, store, or transmit any personal information or user data. All calculations and data processing occur locally on your device.

Permissions Used

The App requests the following permissions:

1. INTERNET Permission

  • Purpose: To fetch live currency exchange rates for the Currency Converter feature
  • Data: Only API calls to currency exchange rate services
  • Storage: No data is stored

2. ACCESS_NETWORK_STATE Permission

  • Purpose: To check network connectivity status
  • Data: No data collected, only checks if device is online/offline

3. WRITE_EXTERNAL_STORAGE Permission

(Android 9 and below only)

  • Purpose: To save exported PDF files to device storage
  • Data: Only files you explicitly export are saved to your device

4. READ_EXTERNAL_STORAGE Permission

(Android 9 and below only)

  • Purpose: To read exported files from device storage
  • Data: Only files you have created are accessed

How We Use Information

Since we do not collect any personal information, we do not use, share, or sell any user data.

Data Storage

All data entered in the App (calculations, denomination entries, etc.) is stored locally on your device using:

  • Android DataStore (for app settings)
  • Local file storage (for exported PDFs)
We do NOT:
  • Store data on external servers
  • Sync data to cloud services
  • Access your data remotely
  • Share data with third parties

Third-Party Services

Currency Exchange Rate API

The Currency Converter feature uses third-party APIs to fetch exchange rates. We do not control these third-party services. Please review their privacy policies if you have concerns.

Children's Privacy

Fynical is suitable for all ages. We do not knowingly collect information from children under 13. Since we do not collect any information, this is not applicable.

Your Rights

Since we do not collect personal information, there is no data to access, modify, or delete. However, you can:

  • Clear app data through Android Settings
  • Uninstall the app at any time
  • Export your calculations before uninstalling

Security

  • All calculations are performed locally on your device
  • No data is transmitted to external servers (except currency API calls)
  • No user accounts or authentication required
  • No personal information is stored

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by:

  • Posting the new Privacy Policy in the App
  • Updating the "Last Updated" date

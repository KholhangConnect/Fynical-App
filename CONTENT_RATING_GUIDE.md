## Content Rating Questionnaire Answers

### 1. Does your app contain user-generated content?
**Answer: No**

**Explanation:** Fynical does not allow users to create, upload, or share content. All calculations are performed locally and results are only stored on the user's device.

---

### 2. Does your app allow users to communicate with each other?
**Answer: No**

**Explanation:** Fynical does not have any communication features such as chat, messaging, comments, or social networking capabilities.

---

### 3. Does your app allow users to share personal information?
**Answer: No**

**Explanation:** While users can export calculation results as PDF/PNG and share them, the app does not facilitate sharing personal information through the app itself. Any sharing is done through standard Android sharing mechanisms (email, messaging apps, etc.) which are outside the app's control.

---

### 4. Does your app contain ads?
**Answer: No**

**Explanation:** Fynical is an ad-free application. No advertisements are displayed within the app.

---

### 5. Does your app allow in-app purchases?
**Answer: No** (or **Yes** if you plan to add premium features)

**Explanation:** Currently, Fynical does not have in-app purchases. If you plan to add premium features in the future, you would answer "Yes" and provide details about the purchase types.

---

### 6. Does your app access location data?
**Answer: No**

**Explanation:** Fynical does not request or use location permissions. The app does not track user location.

---

### 7. Does your app access sensitive user data?
**Answer: No**

**Explanation:** Fynical does not collect, store, or access any sensitive user data such as:

### 8. Does your app contain violence, sexual content, or mature themes?
**Answer: No**

### 9. Does your app contain gambling or betting features?

## Permissions Justification

### INTERNET Permission
- **Purpose:** Fetch live currency exchange rates
- **Data Collected:** None (only API calls, no data stored)
- **User Control:** User can use app offline (currency converter won't work)

### ACCESS_NETWORK_STATE Permission
- **Purpose:** Check if device is online/offline
- **Data Collected:** None
- **User Control:** Informational only

### WRITE_EXTERNAL_STORAGE (Android 9 and below)
- **Purpose:** Save exported PDF files
- **Data Collected:** None (only user's own exported files)
- **User Control:** User explicitly exports files

### READ_EXTERNAL_STORAGE (Android 9 and below)
- **Purpose:** Read exported PDF files
- **Data Collected:** None (only user's own files)
- **User Control:** User's own files only
